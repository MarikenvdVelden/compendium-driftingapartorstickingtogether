# Drifting Apart or Sticking Together? An analysis of party platform changes in 8 Western European countries
Data &amp; Analysis Compendium for the Drifting Apart or Sticking Together? An analysis of party platform changes in 8 Western European countries paper.

# Data
The [integrated data](data/intermediate/cleaned_dyadic_data.csv) integrates, tidies and pre-processes the CMP, ParlGov, Bergmann, Müller and Strøm Cabinet data, and opinion poll data sets. This file is used to test the three hypotheses on the conditions under which parties stick together or drift apart after a term in office. Additionally, the data contains all additional variables to conduct all the robustness checks reported in the Supporting Information. See the scripts in [src/data-processing](src/README.md) for details on how the data is constructed.
